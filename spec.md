# Specification

## Summary
**Goal:** Add a complete Menu page with categorized sections displaying all restaurant dishes.

**Planned changes:**
- Create a new Menu page component with categorized sections (Appetizers, Main Course, Beverages, Desserts)
- Display menu items in card layout with name, description, price, and image
- Add routing for /menu path using TanStack Router
- Update Header navigation to include Menu link between Home and Gallery
- Update Footer quick links to include Menu link
- Add menu item images for all categories

**User-visible outcome:** Users can navigate to a full Menu page from the header and footer, view all available dishes organized by category with images, descriptions, and prices in a luxury-themed layout.
